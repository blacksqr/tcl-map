The nap_land_flag Command

The nap_land_flag command defines land/sea masks. The data has an accuracy of 0.01° of latitude/longitude.

This command is normally used via the functions is_land and is_coast rather than directly.

This command is based on code which was originally written by Dr. Chris. Mutlow at the Rutherford Appleton Laboratory in England. This code has been adapted for NAP by Peter Turner and Harvey Davies.
